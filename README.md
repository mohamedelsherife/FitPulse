FitPulse is a front-end fitness web app built with HTML, CSS, and Vanilla JavaScript.

📄 Pages:
- Login Page – form validation with error/success messages
- Dashboard – welcome screen with dark mode toggle
- Profile – employee card with editable profile form
- Workout – workout selection (Running, Yoga, Gym) with dynamic messages
- Progress Tracker – interactive progress bar with percentage display
- Trainers – trainer cards with details and session booking

🛠️ Tech Stack:
- HTML5
- CSS3 (global stylesheet)
- Vanilla JavaScript (per-page scripts)

✨ Features:
- Shared header & footer via include.js
- Global CSS theme across all pages
- Dark mode support
- Dynamic UI interactions with JavaScript

📁 Project Type: University Project – 2026
